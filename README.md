## ⚙ 应用主题

修改hexo配置文件`_config.yml`，把主题改为`Butterfly`

```
theme: butterfly
```

>如果你沒有pug以及stylus的渲染器，請下載安裝： npm install hexo-renderer-pug hexo-renderer-stylus --save

- [X] 支持二级目录
- [x] 双栏设计
- [x] 响应式主题

- [x] Pjax

- [x] 过期文章提醒
